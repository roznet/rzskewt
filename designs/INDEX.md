# RZSkewT

> Swift package for rendering Skew-T log-P diagrams with full atmospheric thermodynamics

Platform: iOS 18+ / macOS 15+. Depends only on SwiftUI + Foundation.

Package structure, coordinate transform, rendering pipeline, data models, and integration patterns.
